## Synopsis

Python implementation for solving N-queens problem with Hill Climbing Algorithm.

## Code Example

```
game = n_queens.game(n)
game.hill_climb()
```

## Motivation

Done for CS404 - Artificial Intelligence course in Sabanci University in March 2018.
